FAILURE MODE TO RECOGNIZE
If an alert fires, the "monitoring pipeline" step turns red with
  ValueError: monitoring alerts fired: [...]
in its log — but the monitoring-report artifact is STILL uploaded
(if: always()), because the report that explains a failure is most
valuable exactly when there is one.
